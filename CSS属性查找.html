CSS常用属性参考表

全面的CSS属性列表，包含说明和示例值。使用下面的按钮按类别筛选或搜索特定属性。

属性 说明 示例值
display 定义元素的显示类型 block, inline, inline-block, flex, grid, none
position 指定元素的定位方式 static, relative, absolute, fixed, sticky
float 指定元素浮动方向 left, right, none
clear 指定元素哪一侧不允许其他浮动元素 left, right, both, none
flex-direction 指定Flex容器主轴的方向 row, row-reverse, column, column-reverse
justify-content 定义Flex项目在主轴上的对齐方式 flex-start, flex-end, center, space-between, space-around
align-items 定义Flex项目在交叉轴上的对齐方式 stretch, flex-start, flex-end, center, baseline
grid-template-columns 定义Grid布局的列 100px 1fr 2fr, repeat(3, 1fr), minmax(100px, 1fr)
grid-template-rows 定义Grid布局的行 100px 200px, repeat(2, 1fr), auto
grid-gap 定义Grid布局中行和列之间的间隙 10px, 1rem, 5%
属性 说明 示例值
color 设置文本颜色 #ff0000, rgb(255,0,0), red
font-family 指定文本的字体系列 Arial, "Helvetica Neue", sans-serif
font-size 设置文本大小 16px, 1rem, 120%
font-weight 设置文本粗细 normal, bold, 700
text-align 设置文本水平对齐方式 left, right, center, justify
line-height 设置行高 1.5, 24px, normal
text-decoration 设置文本装饰 none, underline, overline, line-through
text-transform 控制文本的大小写 none, capitalize, uppercase, lowercase
letter-spacing 设置字符间距 normal, 2px, 0.1em
word-spacing 设置单词间距 normal, 5px, 0.2em
属性 说明 示例值
background-color 设置元素背景颜色 #ffffff, rgba(0,0,0,0.5), transparent
background-image 设置元素背景图像 url("image.jpg"), linear-gradient(red, yellow)
background-repeat 设置背景图像是否及如何重复 repeat, repeat-x, repeat-y, no-repeat
background-position 设置背景图像的起始位置 left top, center center, 50% 50%
background-size 设置背景图片大小 cover, contain, 100% 100%, 50px 80px
background-attachment 设置背景图像是否固定或者随着页面滚动 scroll, fixed, local
background 所有背景属性的简写 #fff url("image.jpg") no-repeat center
属性 说明 示例值
width 设置元素的宽度 100px, 50%, auto
height 设置元素的高度 200px, 100%, auto
margin 设置元素的外边距 10px, 5px 10px, 5px 10px 15px 20px
padding 设置元素的内边距 10px, 5px 10px, 5px 10px 15px 20px
border 设置元素的边框 1px solid #000, 2px dashed red
border-radius 设置元素边框的圆角 5px, 50%, 10px 5px 15px 20px
box-shadow 设置元素阴影效果 2px 2px 5px #888, inset 0 0 10px #000
box-sizing 定义如何计算元素的宽度和高度 content-box, border-box
overflow 定义内容溢出元素框时的处理方式 visible, hidden, scroll, auto
属性 说明 示例值
opacity 设置元素的不透明度 0.5, 1, 0
visibility 设置元素是否可见 visible, hidden, collapse
transition 设置元素过渡效果 all 0.3s ease, opacity 1s linear
transform 对元素进行旋转、缩放、移动或倾斜 rotate(45deg), scale(1.2), translate(10px, 20px)
animation 设置元素动画效果 fadeIn 2s ease infinite
cursor 设置鼠标指针在元素上的样式 pointer, default, help, move
z-index 设置元素的堆叠顺序 10, -1, auto
filter 定义元素的可视效果（如模糊、饱和度等） blur(5px), grayscale(100%), brightness(150%)
